Welcome to the Messaging Work Bench application help.

The goal of this web page is to give you  a brief introduction to the application. It will be presented as a series of application screen shots in your browser. Be sure that your browser supports frames. It has only been tested with MS Internet Explorer 5.5.

 

Click on the various buttons within the screens to get a description of their purpose and function. Note that the 4 application tabs (Message Definition, Data Sources, Display/Reports and Message Capture)  on the main screen are active, and will display as they do in the real application. Scroll the upper frame to view the lower portions of the window view. 

For operations that don't involve another screen shot, the functional explanation of the button or screen area will appear in the bottom frame of the page. In most cases, the <back> link will simply clear the bottom frame, but in others it will return you to the parent tab on the main application. The border between the Upper and lower frame is resizable (click on it and drag up or down) to accommodate reading the more lengthy descriptions.

Use the application main tabs to navigate back and forth between the major areas of the application. You may also use the web browsers back arrow button if you prefer. In most auxiliary screens, the close button will bring you back to the parent main application tab.
