Profile Export Features

 

Export HL7 Conformance Profile – From the File menu select Export HL7 Conformance Profile.  This option produces the HL7 Conformance Profile in XML. It replaces the two step process formerly required when running the report from the Display/Reports tab.

 

The next two export features Export Profile Package and Export Profile Package to Zip Archive require a ‘one time’ setup step before they can be used. To setup these exports select Options from the Maint menu and select the Misc tab. In the Profile Package Content area indicate which files and reports that you want to become part of the package each time one of these export options is selected from the main menu.

 

File choices for inclusion in the export are the currently opened profile (.mwb file), the active segment libraries, the master table library and data type libraries associated with the segment libraries.

 

You may also select reports from the list to be included in the export package. When reports are selected, the MWB generates the report and copies it to the export archive. User defined reports as well as native MWB reports may be selected for inclusion in the archive. User defined reports may be the only way to produce certain types of reports e.g. the HTML output for the SpecXML must be produced with a user defined report (see the MWB Help article on creating User Defined Reports).

 

Export Profile Related Files to a Disk Folder -  From the File menu select Export Profile Package. You will be presented with a directory browser window. The default is your MWB’s \Projects directory. To add a new subfolder, type it’s name into the folder address window. If you do specify a new folder, you will be asked to verify its creation. Once the folder is selected, all of the files and reports specified in the setup step above will be copied to the folder.

 

Export Profile Related Files to a Zip Archive - From the File menu select Export Prof Pkg to Zip Archive. The files and reports will automatically be copied to a .zip file. When complete, you will be prompted for a filename for the zip archive and folder to save it to.
