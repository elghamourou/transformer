MWB Reports

This page features a list of available reports that can be run and displayed in a window for editing, printing or svaing in various file formats.

Report Editor Tools - These are standard text editing tools for the producing/editing Rich Text Formatted (RTF) documents. These may be used to enhance or modifiy the text produced by invoking on of the standard reports.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Report Selection - This is a list of available reports. There are 2 basic varieties of reports; specification definitions and example messages, each of which is available in plain text or rtf. Message examples can be produced in either ER7 or XML format. Message specifications are available in XML format as well.
